Project Description :

Eco lab is a mobile application that consists of knowledge about eco-enzyme. It includes information about what is eco-enzyme, steps of eco-enzyme production, and eco-enzyme applications. There is thread for dicussing about eco enzyme can there is an eco enzyme calculator for easier eco enzyme making. This mobile app use react native as a frontend and use a laravel via REST API to react native using axios network.
